## ⚙️ Installation & Setup

Clone the repository:

```bash
git clone https://github.com/RISHIKARTHIGAYAN/Portfolio.git
cd Portfolio
```

Install dependencies:

```bash
npm install
```

Run locally:

```bash
npm run dev
```

Open in browser:

```
http://localhost:5173/portfolio/
```
